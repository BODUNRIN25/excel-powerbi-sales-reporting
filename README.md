# Excel–Power BI Sales Reporting System

## 📌 Overview
This project is an integrated Excel and Power BI solution designed to streamline sales recording, automate invoice and receipt generation, and deliver clean, analysis‑ready data to interactive Power BI dashboards. It offers a lightweight alternative to expensive ERP systems while ensuring accuracy, consistency, and actionable business insights.

---

## ❗ The Problem
Many small and medium‑sized businesses rely on manual spreadsheets, WhatsApp messages, or scattered files to track sales and payments. This leads to:
- Frequent billing errors  
- Lost or inconsistent records  
- Slow month‑end reporting  
- Limited visibility into revenue performance  
- Difficulty generating professional invoices/receipts  

---

## ✅ What This System Solves
- Centralizes all sales entries in one validated Excel table  
- Automatically generates branded invoices and receipts (PDF)  
- Ensures consistent pricing using SKU lookups  
- Simplifies payment tracking  
- Eliminates duplicated or incorrect entries  
- Feeds structured data directly into a Power BI reporting model  
- Enables real‑time visibility of sales performance  

---

## ✨ Features
- **Sales Entry Table:** Dropdowns, data validation, auto‑calculated totals  
- **Invoice & Receipt Automation:** Auto‑filled templates with one‑click PDF export  
- **SKU/Product Lookup:** Consistent product descriptions and pricing  
- **Payment Log:** Tracks customer payments and methods  
- **Power BI Reporting:** Revenue trends, top products, MoM growth, customer insights  
- **Power Query ETL:** Cleans and prepares data for reporting  
- **Professional Outputs:** Clean, branded documents ready for clients
